# Chatbot_Rasa

### To run the chatbot cli:
- Run each line in seperate terminal
```
rasa run actions
docker run -p 8000:8000 rasa/duckling
rasa shell
````

### To run the chatbot in a browser
- Run each line in seperate terminal
```
rasa run actions
docker run -p 8000:8000 rasa/duckling
rasa x
```
